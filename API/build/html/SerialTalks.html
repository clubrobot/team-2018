SerialTalks¶

La librairie SerialTalks assure la comunication entre une ordinateur et un arduino. On trouve dans cette librairie deux versions asymétriques pour chaque terminal.

Python Object¶

Pour ouvrir les comunications avec un arduino, il faut creer un object SerialTalks et lancé la connexion avec les méthodes disponibles de celle-ci.

Object SerialTalks¶

class SerialTalks¶
__init__(port)¶
Paramètres:port – Adresse de l’arduino a connecter, le format de l’adresse dépend de l’OS utilisé. Sous Windows elle sera de la forme : COM3 , Alors que sous Linux on aurra une adresse de la forme : /dev/ttyUSB0 ou /dev/arduino/WheeledBase
connect(timeout=5)¶

Cette methode permet de connecter l’arduino avec l’ordinateur, si cette méthode s’execute sans erreur, il est ensuite possible d’envoyer des instructions

Paramètres:

timeout (int) – Choisi un timeout pour la connexion entre l’ordinateur et l’arduino (en seconde)
Lève:
  • ConnectionFailedError – Dans le cas ou l’arduino n’est pas trouvé ou ne peux pas être configuré.
  • AlreadyConnectedError – Dans le cas ou l’arduino est déjà connecté.
disconnect()¶

Permet de déconnecter l’arduino.

send(opcode, *args)¶

Permet l’execution d’une requette sans retour d’information de la part de l’arduino, exemple définir une nouvelle position.

Paramètres:
  • opcode (hexa) – Code correspondant à la requette demandé. Généralement utilisé sous forme de nombre en hexadécimal ; example : 0xF4
  • args (bytes) – Arguments à ajouter en fonction de la requette executé, toujours sous la forme de bytes. Pour la conversion utilisé SerialUtils.
Retourne:

Code d’indentification pour le retour des informations (retcode). Nombre entier généré aléatoirement entre 0 et 4294967295.
get_queue(retcode)¶
Paramètres:retcode (int) – Code d’indentification donné lors de l’envoie de la requette pour avoir le retour de l’arduino a propos de cette meme requette.
Retourne:Renvoie l’object Queue relié avec le retcode.
delete_queue(retcode)¶

Permet la suppression d’une Queue

Paramètres:retcode (int) – Code d’indentification de la queue à supprimer.
reset_queues()¶

Permet la suppression de toutes les Queues .

poll(retcode, timeout=0)¶

Methode pour récuper un message en attente dans une queue.

Paramètres:
  • retcode (int) – Code d’indentification de la queue à utiliser.
  • timeout (int) – Timeout pour la reception du message.
Retourne:

Message en bytes.
flush(retcode)¶

Méthode pour vider une queue

Paramètres:retcode (int) – Code d’indentification de la queue à utiliser.
execute(opcode, *args, timeout=5)¶

Methode pour executé une requette avec un retour.

Paramètres:
  • opcode (int) – Code d’indentification de la requette à effectuer
  • args (bytes) – Argument à transmettre à l’arduino, attention les convertir en bytes avant ( voir Object de SerialUtils)
  • timeout (int) – Timeout de la reception (en seconde).
Retourne:

Arguments recu de l’arduino sous l’object Deserialser voir Object de SerialUtils
getuuid(timeout=5)¶

Methode pour demandé à l’arduin son indentification

Paramètres:timeout (int) – Timeout de la reception (en seconde) de l’indentification.
Retourne:L’indentification de l’arduino
setuuid(uuid)¶

Methode pour définir un nouvelle indentification pour l’arduino

Paramètres:uuid – Nouvelle indentification pour l’arduino
